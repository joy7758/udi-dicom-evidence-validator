# v0.4.0 Roadmap

v0.4 focuses on release hardening, external review readiness, and remote
reproducibility for the public validator.

Planned public-only surfaces:

- clean clone smoke testing
- remote GitHub release verification
- public release asset builder
- external review brief and checklist
- citation and archive readiness metadata
- public GitHub Actions for reproducibility checks

Out of scope:

- real patient data
- raw DICOM samples
- clinical validation
- regulatory approval
- certification
- private conformance suite exposure
- private sample validation service exposure
