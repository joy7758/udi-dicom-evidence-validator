# Paper Finalization Gate v0.8

Current gate: `PAPER_FINALIZATION_READY_PENDING_HUMAN_REVIEW`

Required human checks:

- title
- target journal
- final abstract
- reference verification
- figure rendering
- DOI insertion after Zenodo record exists

The v0.8 paper finalization bundle is generated by:

```bash
python scripts/build_paper_finalization_bundle.py
python scripts/check_paper_finalization_bundle.py artifacts/paper-finalization-bundle-v0.8
```

The bundle is Markdown and JSON only. It does not generate Word or PDF files,
does not fabricate references, and does not fabricate DOI metadata.

Boundary remains no PHI, no raw DICOM, not clinical validation, not regulatory
approval, not certification, Device UID != UDI-DI, offline fixture first, live
openFDA explicit opt-in, FDO-style mapping only, and no robot operation
evidence.
