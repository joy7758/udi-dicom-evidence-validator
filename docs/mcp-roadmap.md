# MCP Roadmap

Future public tools may include `list_profile`, `list_public_examples`,
`validate_public_manifest`, `render_public_receipt`, and
`explain_failure_code`.

Future private tools, if ever implemented in a controlled partner environment,
may include `run_partner_conformance_suite`, `compare_against_golden_output`,
`generate_partner_report`, and `map_vendor_dicom_fields`. They are not
implemented in this public repository.
