# Reviewer Feedback Triage

Triage order:

1. Identify whether the report is reproducible from public synthetic examples.
2. Check whether it concerns documentation, schema clarification, validator
   behavior, or boundary language.
3. Reject or redirect requests that require PHI, raw DICOM, private suite
   exposure, clinical validation, regulatory approval, or certification.
4. Convert accepted issues into small, public-safe tasks.

Do not treat reviewer feedback as proof of clinical validity or regulatory
status.
