# v0.4.0 Roadmap

v0.4 focuses on release hardening, external review readiness, and remote
reproducibility for the public validator.

Planned public-only surfaces:

- clean clone smoke testing
- remote GitHub release verification
- public release asset builder
- external review brief and checklist
- citation and archive readiness metadata
- public GitHub Actions for reproducibility checks

Out of scope:

- real patient data
- raw DICOM samples
- clinical validation
- regulatory approval
- certification
- private conformance suite exposure
- private sample validation service exposure

Next review stage:

- v0.5 prepares public paper review material, reviewer quickstart/checklists,
  and a reproducibility capsule.
- v0.5 does not add new medical capability, real samples, robot operation
  evidence, clinical validation, regulatory approval, or certification.
- v0.6 prepares journal-neutral paper submission readiness, DOI-ready archive
  metadata review, and external feedback intake without changing validator
  behavior or claiming an assigned DOI.
