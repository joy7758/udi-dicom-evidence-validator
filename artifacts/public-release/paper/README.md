# Paper Review Pack v0.5

This directory is a public-safe paper drafting and external review pack for the
UDI-DICOM Evidence Validator. It contains a manuscript outline, section drafts,
figures, and tables that describe the public validator, synthetic examples, and
reproducibility capsule.

Scope:

- Minimal UDI-DICOM evidence manifest and deterministic validator.
- Public synthetic examples only.
- Offline registry fixture first; live openFDA remains explicit opt-in.
- FDO-style mapping as metadata alignment only, not an official FDO standard
  implementation.

Out of scope:

- PHI, raw DICOM files, real hospital samples, real customer data.
- Clinical validation, regulatory approval, certification, diagnosis, safety
  assurance, or hospital deployment claims.
- Medical robot operation evidence.
