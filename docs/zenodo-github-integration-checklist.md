# Zenodo GitHub Integration Checklist

- [ ] Confirm the public GitHub repository is selected for archiving.
- [ ] Confirm private suite and private service repositories are not selected
      for public DOI deposition.
- [ ] Confirm GitHub Release assets are public-safe.
- [ ] Review `.zenodo.json`.
- [ ] Review `codemeta.json`.
- [ ] Review `CITATION.cff`.
- [ ] Confirm no DOI is claimed before Zenodo assigns one.
- [ ] Confirm release notes include no PHI, no raw DICOM, not clinical
      validation, not regulatory approval, and not certification.
