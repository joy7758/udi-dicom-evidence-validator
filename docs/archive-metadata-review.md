# Archive Metadata Review

Archive metadata review checks:

- Title consistency.
- Author consistency.
- Repository URL consistency.
- License consistency.
- Keyword coverage.
- Version field presence.
- DOI-ready wording without claiming an assigned DOI.
- Public repository scope only.

Run `scripts/build_archive_metadata_report.py` and
`scripts/check_archive_metadata_report.py` before an archive integration step.
