# v0.7.0 Public Release Plan

v0.7.0 is a paper-submission preparation release. It does not introduce new
validator functionality.

Release readiness requires:

- v0.6 paper submission package checks pass.
- v0.6 archive metadata checks pass.
- public release assets remain public-only.
- reproducibility capsule checks pass.
- pytest, ruff, and mypy pass.
- release notes preserve no PHI, no raw DICOM, not clinical validation, not
  regulatory approval, not certification, no fake DOI, Device UID != UDI-DI,
  offline fixture first, live openFDA explicit opt-in, FDO-style mapping only,
  and no robot operation evidence.

Do not create a DOI claim in this repository. DOI assignment remains an
archive-service outcome after human review.
