# v0.3.0 Roadmap

v0.3.0 focuses on release hardening, reproducible evaluation, agent-readable
interfaces, and regression harnesses.

Planned scope:

- Golden receipt regression for public v0.1 and v0.2 fixtures.
- Public evaluation matrix for synthetic examples.
- Release artifact inventory and reproducibility checklist.
- Public-only MCP skeleton for read-only agent access.
- Compatibility matrix for profile and validator versions.

Out of scope:

- Clinical validation.
- Regulatory approval.
- Certification.
- Robot operation evidence.
- PHI processing.
- Private conformance suite access.
