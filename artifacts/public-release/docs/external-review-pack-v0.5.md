# External Review Pack v0.5

This review pack gives an external technical reviewer enough public material to
inspect the validator boundary, run the synthetic examples, compare golden
receipts, and check the reproducibility capsule.

Review order:

1. Read `docs/reviewer-risk-boundary-v0.5.md`.
2. Run `docs/reviewer-quickstart-v0.5.md`.
3. Compare outputs with `docs/reviewer-expected-outputs-v0.5.md`.
4. Record findings using `docs/reviewer-reproduction-log-template.md`.
5. Use `docs/reviewer-checklist-v0.5.md` for final review status.

The pack is public-only. It does not include private conformance cases, private
service workflows, PHI, raw DICOM files, real hospital samples, or real customer
data.
