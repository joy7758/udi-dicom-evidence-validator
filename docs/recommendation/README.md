# Department Recommendation Pack

This directory contains public, boundary-safe materials for department-level
recommendation, technical review, and internal handoff discussions.

Core documents:

- `department-recommendation-report-cn.md` for a structured Chinese report.
- `executive-one-page-cn.md` for a one-page decision brief.
- `evidence-map.md` for the public evidence chain.
- `boundary-and-risk-statement-cn.md` for explicit limitation wording.
- `attachment-checklist.md` for safe attachments.
- `email-templates-cn-en.md` for concise outreach drafts.
- `meeting-script-cn.md` for a short review meeting script.
- `next-step-gate.md` for follow-up gates.

Boundary keywords for automated review: no PHI, no raw DICOM, not clinical
validation, not regulatory approval, not certification, public synthetic
examples, Device UID != UDI-DI.

The pack references:

- Public repository: https://github.com/joy7758/udi-dicom-evidence-validator
- Verified Zenodo DOI: https://doi.org/10.5281/zenodo.20540532
- Paper DOI: https://doi.org/10.1007/s10278-026-02019-6

This pack is a recommendation and reproducibility artifact. It is not a
commercial proposal, deployment commitment, clinical validation package,
regulatory approval package, or certification claim.
