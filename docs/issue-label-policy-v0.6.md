# Issue Label Policy v0.6

Recommended labels:

- `reproducibility`
- `documentation`
- `boundary-concern`
- `schema-clarification`
- `validator-bug`
- `out-of-scope`
- `needs-human-review`

Boundary concerns must be reviewed before any implementation change. Out-of-
scope requests include real hospital pilots, real device samples, robot
operation evidence, clinical validation, regulatory approval, certification, and
commercial packaging.
