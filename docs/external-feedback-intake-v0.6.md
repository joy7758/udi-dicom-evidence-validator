# External Feedback Intake v0.6

External reviewer feedback should be filed through issue templates and triaged
into one category:

- reproducibility issue
- documentation issue
- boundary concern
- schema clarification
- validator bug
- out-of-scope request

Do not accept PHI, raw DICOM files, real patient data, real hospital data, real
device serial numbers, regulatory certification requests, or clinical
validation requests through public issues.
